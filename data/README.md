# Dataset (Not Included)

This repository **does not include any datasets**.

All experiments in this project were conducted using a **public CT segmentation dataset**.  
Due to licensing and size constraints, users must download and prepare the dataset separately.

---

## Dataset Source

**Chest CT Segmentation (lung, heart, trachea)**  
https://www.kaggle.com/datasets/polomarco/chest-ct-segmentation

The dataset consists of volumetric chest CT scans with voxel-wise annotations for three anatomical structures.
